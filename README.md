# 🦅 RCC IntelOps — Central de Instrução Inteligente

> Sistema Web com Assistente Virtual Integrado (`supervisorbot`) desenvolvido para consulta ágil de diretrizes doutrinárias, regulamentos e segurança com base na Apostila de Auditoria da Polícia RCC.

---

## 💡 Sobre o Projeto

Este projeto foi concebido como um **Protótipo de Validação de Conceito**. O objetivo principal foi atender aos requisitos básicos e estruturais do edital/apostila para demonstrar viabilidade técnica. 

O motor de busca do chat utiliza um algoritmo local de cruzamento semântico de radicais (tokens). Embora funcional e preciso para o escopo proposto, vale ressaltar que **em um cenário real e de alta performance, este assistente pode ser integrado a arquiteturas avançadas de Inteligência Artificial**, como:
* Modelos de Linguagem de Grande Porte (LLMs) via API (OpenAI, Gemini, Anthropic).
* Mecanismos de RAG (Retrieval-Augmented Generation) acoplados a Bancos de Dados Vetoriais (Pinecone, ChromaDB) para leitura dinâmica de PDFs complexos sem necessidade de catalogação manual.

---

## 🎯 Itens da Apostila Atendidos

O sistema cobre os tópicos essenciais de conformidade descritos na documentação oficial, divididos em:

*   **Gestão de Carreira:** Critérios diferenciais para promoção (de Soldado a Tenente) e aplicação correta de Diálogos Táticos (**DPP** e **DPR**).
*   **Regulamento Disciplinar:** Processos administrativos de rebaixamento e demissão, autonomia de patentes (Subtenentes) e penalidades contra nepotismo (CPM).
*   **Segurança da Informação:** Identificação de contas fakes (padrões de nicks), segurança de credenciais pessoais e protocolos de emergência para recuperação/congelamento de contas hackeadas (General+/Superintendente+).
*   **Logística e Operações:** Validação de permissões no RCCSystem via capturas de tela (prints em Windows, Mac e Mobile), hospedagem de evidências (Imgur) e verificação de segurança no Batalhão (Postos de defesa e IDs de donos de emblemas).
*   **Integração de Sistemas:** Cadastro e ativação síncrona entre o Fórum da RCC e o RCC System através de códigos inseridos na missão do Habbo.

---

## 💬 O que você pode perguntar para a IA?

O `supervisorbot` foi calibrado para responder de forma inteligente mesmo a variações de frases. Abaixo estão exemplos práticos de perguntas que você pode testar no widget do chat:

### 🛡️ Emergências e Segurança
* *"Minha conta foi hackeada o que fazer?"*
* *"Como identificar um policial fake?"*
* *"Como proteger a minha senha do Habbo?"*

### 📈 Promoções e Diálogos
* *"Quais são os requisitos para promover de aspirante a tenente?"*
* *"O que falar em um diálogo pós-rebaixamento (DPR)?"*
* *"Me fale sobre a dpp após promoção"*

### ⚖️ Infrações e Regras
* *"O que acontece se eu cometer nepotismo?"*
* *"Um subtenente pode punir ou rebaixar alguém sozinho?"*
* *"Quais as 6 punições administrativas?"*

### 📸 Prints e Sistemas
* *"Como tirar print no celular ou no windows?"*
* *"Qual o site correto para hospedar imagens?"*
* *"Como faço para ativar a minha conta no RCC System?"*
* *"Quem são os donos oficiais dos emblemas da RCC?"*

---

## 🛠️ Tecnologias Utilizadas

O projeto foi construído utilizando tecnologias nativas da web para garantir máxima leveza, compatibilidade e velocidade de carregamento (Zero dependências externas):

*   **HTML5:** Estruturação semântica da aplicação e do widget de chat.
*   **CSS3 Moderno:** Design responsivo com visual *Cyberpunk/Glassmorphism*, efeitos de iluminação Neon e animações de pulsação de anel (utilizando a imagem `professores.png` de forma exclusiva no gatilho do chatbot).
*   **JavaScript (ES6+):** Motor assíncrono de manipulação de DOM, simulador de digitação e algoritmo anticonflito que separa mensagens contextuais de saudações casuais.
